# Sistema de Controle Financeiro

Aplicação web desenvolvida com Python, Flask e JavaScript para auxiliar no gerenciamento de receitas e despesas, permitindo o acompanhamento e controle das movimentações financeiras.

## Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Cadastro de receitas e despesas
- Listagem de movimentações financeiras
- Edição de registros
- Interface simples e intuitiva
- Atualização dinâmica com JavaScript
- Controle básico de finanças pessoais

## Objetivo

Este projeto foi desenvolvido durante o curso Técnico em Análise e Desenvolvimento de Sistemas no SENAI com o objetivo de praticar:

- Desenvolvimento web com Flask
- Manipulação de formulários
- Integração entre Python e JavaScript
- Estruturação de aplicações web
- Operações CRUD
- Organização de arquivos estáticos

## Aprendizados

Durante o desenvolvimento foram aplicados conceitos de:

- Rotas Flask

- Organização de interfaces web
