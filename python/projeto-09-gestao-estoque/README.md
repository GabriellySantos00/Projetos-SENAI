# Sistema de Gestão de Estoque

Sistema web desenvolvido com Python e Flask para gerenciamento de estoque, permitindo controlar produtos, quantidades e movimentações de forma simples e organizada.

## Tecnologias Utilizadas

- Python
- Flask
- HTML5
- Excel (OpenPyXL/Pandas)
- CSS3

## Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Atualização de quantidades
- Visualização dos registros
- Importação e exportação de dados em planilhas
- Interface web para gerenciamento

## Objetivo

Este projeto foi desenvolvido durante o curso Técnico em Análise e Desenvolvimento de Sistemas no SENAI com o objetivo de praticar:

- Desenvolvimento web com Flask
- Manipulação de planilhas Excel
- Gerenciamento de dados
- Estruturação de aplicações web
- Organização de informações de estoque

## Aprendizados

Durante o desenvolvimento foram aplicados conceitos de:

- Rotas Flask
- Templates HTML
- Leitura e escrita de arquivos Excel
- Manipulação de dados com Python
- Controle de estoque
- Organização de aplicações web
