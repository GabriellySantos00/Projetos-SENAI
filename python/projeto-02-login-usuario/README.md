# Sistema de Login e Cadastro de Usuários

Aplicação web desenvolvida com Python e Flask para gerenciamento de usuários, permitindo cadastro, autenticação, edição e listagem de registros armazenados em arquivo JSON.

## Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3
- JSON

## Funcionalidades

- Cadastro de usuários
- Login de usuários
- Listagem de usuários cadastrados
- Edição de dados dos usuários
- Armazenamento local em arquivo JSON
- Interface web simples e funcional


## Objetivo

Este projeto foi desenvolvido durante o curso Técnico em Análise e Desenvolvimento de Sistemas no SENAI com o objetivo de praticar:

- Desenvolvimento web com Flask
- Manipulação de arquivos JSON
- Criação de formulários
- Rotas e templates
- Operações CRUD
- Estruturação de aplicações web

## Aprendizados

Durante o desenvolvimento foram aplicados conceitos de:

- Framework Flask
- Leitura e escrita em arquivos JSON
- Organização de templates HTML
- Processamento de formulários
- Validação de dados
- Navegação entre páginas
- Gerenciamento de usuários
