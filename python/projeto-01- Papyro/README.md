# Papyro - Sistema de Biblioteca

Sistema web desenvolvido com Flask e MySQL para gerenciamento de bibliotecas, permitindo o cadastro e controle de leitores, livros e empréstimos.

## Tecnologias Utilizadas

- Python
- Flask
- MySQL
- HTML5
- CSS3

## Funcionalidades

### Leitores
- Cadastrar leitores
- Listar leitores
- Editar informações dos leitores

### Livros
- Cadastrar livros
- Listar livros
- Editar informações dos livros

### Empréstimos
- Realizar empréstimos
- Listar empréstimos
- Editar empréstimos registrados


## Objetivo

Este projeto foi desenvolvido durante o curso Técnico em Análise e Desenvolvimento de Sistemas no SENAI com o objetivo de praticar:

- Desenvolvimento web com Flask
- Integração com banco de dados MySQL
- Operações CRUD
- Relacionamento entre entidades
- Organização de projetos em camadas
- Estruturação de sistemas de gestão

## Aprendizados

Durante o desenvolvimento foram aplicados conceitos de:

- Rotas Flask
- Templates HTML
- Conexão com banco de dados
- Manipulação de formulários
- Consultas SQL
- Atualização e exclusão de registros
- Controle de empréstimos em sistemas de biblioteca
