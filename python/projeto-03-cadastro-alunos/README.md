# Sistema de Cadastro de Alunos

Aplicação web desenvolvida com Python e Flask para gerenciamento de alunos, permitindo cadastrar, visualizar, editar e remover registros através de uma interface simples e intuitiva.

## Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3

## Funcionalidades

- Cadastro de alunos
- Listagem de alunos cadastrados
- Edição de informações
- Exclusão de registros
- Interface web responsiva
- Organização em páginas separadas para cada operação

## Objetivo

Este projeto foi desenvolvido durante o curso Técnico em Análise e Desenvolvimento de Sistemas no SENAI com o objetivo de praticar:

- Desenvolvimento web com Flask
- Estruturação de aplicações em Python
- Manipulação de formulários
- Operações CRUD
- Organização de templates e arquivos estáticos
- Navegação entre páginas

## Aprendizados

Durante o desenvolvimento foram aplicados conceitos de:

- Rotas Flask
- Templates HTML
- Formulários web
- Métodos HTTP (GET e POST)
- Estrutura MVC simplificada
- Organização de projetos web
- Interface de gerenciamento de dados
