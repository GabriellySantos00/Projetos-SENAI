# Quitanda do Tomate Cru

Sistema web desenvolvido com Python, Flask e JavaScript para gerenciamento de produtos de uma quitanda, permitindo cadastrar, visualizar, editar e excluir itens de forma prática e organizada.

## Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- MySQL

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos cadastrados
- Atualização de informações
- Exclusão de registros
- Integração com banco de dados
- Requisições assíncronas utilizando JavaScript
- Interface simples e intuitiva

## Objetivo

Este projeto foi desenvolvido durante o curso Técnico em Análise e Desenvolvimento de Sistemas no SENAI com o objetivo de praticar:

- Desenvolvimento web com Flask
- Integração com banco de dados MySQL
- Operações CRUD
- Consumo de dados com JavaScript
- Manipulação de formulários
- Organização de aplicações Full Stack

## Aprendizados

Durante o desenvolvimento foram aplicados conceitos de:

- Rotas Flask

- Integração Front-end e Back-end
- Banco de dados relacional
- Estruturação de projetos web
