# Jogo de Corrida

Jogo de corrida desenvolvido com Python, Flask, HTML, CSS e JavaScript. O objetivo é desviar dos carros inimigos e sobreviver o maior tempo possível, acumulando pontos durante a partida.

## Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Controle do carro do jogador
- Movimentação lateral utilizando o teclado
- Geração de carros inimigos
- Sistema de colisão
- Contador de pontos
- Interface visual personalizada com sprites

## Objetivo

Evitar os carros inimigos pelo maior tempo possível e alcançar a maior pontuação.

## Aprendizados

Durante o desenvolvimento deste projeto foram praticados conceitos de:

- Manipulação do DOM com JavaScript
- Eventos de teclado
- Animações e movimentação de elementos
- Detecção de colisões
- Organização de projetos Flask
- Integração entre Python e Front-end
