# Catálogo de Filmes

Sistema web desenvolvido com Flask para gerenciamento de um catálogo de filmes, permitindo cadastrar, visualizar, editar e excluir registros armazenados em banco de dados MySQL.

## Tecnologias Utilizadas

- Python
- Flask
- MySQL
- HTML5
- CSS3

## Funcionalidades

- Cadastro de filmes
- Listagem de filmes cadastrados
- Edição de informações
- Exclusão de registros
- Interface web simples e intuitiva
- Integração com banco de dados MySQL


## Objetivo

Este projeto foi desenvolvido durante o curso Técnico em Análise e Desenvolvimento de Sistemas no SENAI, com o objetivo de praticar o desenvolvimento de aplicações web utilizando Flask, integração com banco de dados e operações CRUD.
