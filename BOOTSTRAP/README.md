# Bootstrap SG

Projeto desenvolvido durante as aulas de Bootstrap no SENAI, com o objetivo de praticar conceitos de layout responsivo, sistema de grid, componentes e customização de estilos.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- Sass

## Conteúdo

- Sistema de Grid
- Layout Responsivo
- Navbar
- Accordion
- Forms
- Badges
- Customização com Sass

## Estrutura

- bs_layout.html
- bs_grid.html
- bs_components.html
- bs_download.html
- custom.scss

## Objetivo

Praticar a utilização do Bootstrap na construção de interfaces responsivas e organizadas.
